# SalonEase Images

This folder contains static images for the SalonEase application.

## Recommended Images

### Logo & Branding
- `logo.png` - SalonEase logo (suggested size: 200x50px)
- `favicon.ico` - Favicon icon (16x16px or 32x32px)

### UI Elements
- `salon-bg.jpg` - Background image for hero section
- `service-placeholder.jpg` - Default service image
- `staff-placeholder.jpg` - Default staff profile image

### Icons
- `calendar-icon.png` - Calendar/booking icon
- `user-icon.png` - User profile icon
- `service-icon.png` - Services icon
- `staff-icon.png` - Staff management icon

## Image Guidelines

- Use optimized images (WebP format preferred)
- Maintain consistent style and branding
- Ensure accessibility with proper alt text
- Keep file sizes small for better performance
